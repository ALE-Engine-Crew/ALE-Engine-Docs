¿Qué es ALE Engine?

ALE Engine es la evolución definitiva para la creación de mods de Friday Night Funkin'. Diseñado por modders para modders, nuestro motor combina potencia, flexibilidad y facilidad de uso en un paquete optimizado.

🚀 Características Principales

⚡

Rendimiento Optimizado

Motor completamente reescrito para ofrecer el máximo rendimiento en PC y móviles, con soporte para hardware antiguo y nuevas tecnologías.

🎮

Scripting Avanzado

Soporte completo para Lua y HScript con APIs documentadas y ejemplos prácticos para modificar cualquier aspecto del juego.

📱

Multiplataforma

Diseñado pensando en la compatibilidad móvil desde el principio, con controles adaptables y optimización para diferentes pantallas.

👨‍💻 Para Desarrolladores

ALE Engine ofrece un entorno de desarrollo intuitivo y potente, diseñado para agilizar tu flujo de trabajo y permitirte concentrarte en lo que realmente importa: tu creatividad.

✔ API completa para Lua/HScript
🔧 Herramientas de depuración
⚙️ Sistema de eventos robusto
🚀 Hot-reloading de assets
